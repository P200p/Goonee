# Product Overview

Goonee Terminal is a hacker-themed Progressive Web Application (PWA) that provides a browser-based terminal interface with cybersecurity tools and educational labs.

## Core Features
- **Terminal Interface**: Matrix-style animated background with green hacker aesthetic
- **Network Scanner**: Tools for network scanning and vulnerability detection
- **Encryption Tools**: AES-256 and RSA encryption/decryption utilities
- **System Monitor**: Real-time system status monitoring
- **Proxy Manager**: VPN and proxy management tools
- **Educational Labs**: Interactive cybersecurity learning modules
- **SharkTool**: Browser extension toolkit for web analysis and modification

## Target Audience
- Cybersecurity students and professionals
- Ethical hackers and penetration testers
- Web developers learning security concepts
- Anyone interested in browser-based security tools

## Key Components
- **Main Terminal**: Central hub with search, shortcuts, and command execution
- **LAB Module**: Interactive cybersecurity exercises (gambling API simulation, classical ciphers, network forensics)
- **SharkTool**: Modular browser toolkit with theme management and monitoring capabilities
- **Scan Module**: Automated scanning and analysis tools

## Language Support
Primary language is Thai with English support. All UI text and documentation should maintain bilingual capability.
